CSVmaker

v1.0.0

A lightweight Python tool for generating CSV files from your local directory structure. Point it at any folder and it recursively scans sub-directories, capturing full file paths into a clean CSV. Great for managing large collections of files — pictures, videos, documents, game assets — you name it.

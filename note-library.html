Adding Notes — Quick Reference

Drop files into a workspace folder and push. That's it — the library auto-discovers everything.

  1. Create your note as .html or .md (Markdown).
  2. Place it in the appropriate workspace folder under notes/:
    notes/sims-guides/ · notes/tools/ · notes/projects/
  3. To create a new workspace, just add a new folder inside notes/.
  4. Commit and push — the Note Library reads from GitHub automatically.

Filenames become display titles: hyphens & underscores are replaced with spaces, and each word is capitalized. Use descriptive filenames like Sims1-install-guide.html.
